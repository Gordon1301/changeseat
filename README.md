78 為晚宴好好打扮

"好吧,那就這樣吧。"

我不由得點了點頭。

我想這個世界上再也沒有人能比得上我的媽媽,她是世界上最好的!

"媽媽!"我大聲的喚她。

"艾莎?"媽媽很疑惑,她不明白我為什麼突然這樣做。

我沒有迴應媽媽,而是一頭扎進了她的懷抱。

意料之中的,我聞到了媽媽身上淡淡的香味,讓我覺得十分温暖。

"怎麼突然撒嬌了?"

媽媽歪頭看着我。

我在媽媽的懷裏用悶悶的聲音回答:"因為我覺得自己的媽媽太棒了呀。"

媽媽還是不懂我在説什麼,一臉訝異地看着我。

可是我沒有再解釋了,因為現在,我已經很安心了。

是阿,要想成為埃爾米帝國的國母和皇后,至少要做得和母親一樣好。

不然我無論如何也不同意著。

心裏這麼想着,心情也就好了不少。我帶着愉快的情緒回到了自己的宮殿裏。

我叉着腰,對我的侍女點點頭,告訴她們:

"今天晚上我要成為這場宴會上最好看的人,請你們幫我打扮一下吧。"

幾個侍女面面相覷,目瞪口呆的看着對方,好像聽到了什麼不可思議的話。

她們已經忙碌了一整天,我回來之後突然給她們帶來一個這樣的命令,不由得讓她們格外驚訝了。

"我想成為這次宴會上最漂亮的人,所以希望你們把我打扮的比平時更漂亮。"

我又解釋了一遍。

她們好像突然領悟到了我話裏的意思,互相望了一眼,雖然有些苦惱和不知所措,但還是點了點頭。

....阿爾米尼亞公主!

感受到了心中的鬥志在熊熊燃燒着,我告訴自己,要用最完美的狀態告訴她:你配不上我的哥,埃爾米帝國的皇太子殿下!

我的眼眶有一點點發熱,侍女看到我的樣子,慌得梳子都握不穩了,險些掉到了地上,引起了一些小小的騷動。

我身上穿着一條閃爍着微微光澤的紫丁香連衣裙,背後還自然的繫着一個藍色的蝴蝶結。配上一條閃閃發光的鑽石項鍊和手鐲,再在餐角戴上一個鑽石頭飾。

侍女將我濃密的頭髮編成了一個小辮子,蓬鬆的銀色髮辮就像珠寶一樣在太陽的照耀下閃爍着光芒。

我目不轉睛地盯着鏡子裏的自己,看着女僕們的手在我的頭上嫺熟的跳躍着,多虧了她們熟練的精心打扮,我感覺今天的自己格外光彩照人。

但是為了確認自己在別人的眼中是否也是這樣美麗,我扭頭問身後的侍女們:

"你們覺得我今天的打扮怎麼樣?"

她們沉默了一瞬間,才驚呼着開口:

"太美了!"

"像精靈!"

"公主殿下是真正的天使,是天上降落的天使!"

侍女們狂熱的反應,更加堅定了我的自信心。

我是知道她們平時對我的打扮的反應程度的,如今反應得這麼熱烈,看來效果還不錯。

我現在對自己的裝扮充滿了自信。

"好吧,現在要準備出門了,時間差不多了。"

我看了一下鐘錶,到了和阿爾迪王子約定的時間了。

我走出房間,果然看到他在我宮殿附近坐着,他正抬頭看着天上的雲,看起來好像等了有一會兒了。

"阿爾迪王子。"

我用略顯低沉的聲音叫了他一聲,然後開始不動聲色色的打量他今天的裝扮。

他穿着米白色的西裝,暗綠色的背心,戴着白手套。把頭髮梳理的整整齊齊,顯得整個人很乾練很精神。

西裝外套口袋裏掛着的金錶鏈,讓他整個人更增添了一分古典的魅力。

乍一看,好像還挺不錯的樣子。我走到他面前,輕輕的提起裙襬行了一個禮,權當是打招呼。

"今天的晚宴,就拜託王子殿下了,請多多關照。"

但是出人意料的,他的反應反倒顯得有些異常,格外緩慢。

"...好的,沒問題。"

面對這樣的他,我只能微微驚訝的看他一眼,詢問道:

"還有什麼問題嗎?"

"不,不是。"

還是像下午一樣,他面帶微笑的看了我一眼。

"不過..."

"....?"

"因為你的模樣太美了。"他誇讚道。

臉頰有一些熱氣上湧,我想現在的自己,臉肯定紅得不行。

但是我也不是一個善於交際的人,面對他的誇讚,我也只能微笑着説:"謝謝你,阿爾迪王子。"

"沒什麼,但是,艾莎公主..."

他好像想説些什麼的模樣,欲言又止的。

"其實也沒什麼,我就是想説..."

"想説什麼?"

他用紫色的眼睛深深地望着我。

"公主,如果我叫你的名字,你會不會覺得我失禮?當然,我希望公主也能叫我阿爾迪。"

"啊?"

我眨了眨眼睛,在社交圈裏,互相稱呼名字是十分親密的行為,很顯然,我們兩個的關係還遠遠不到可以互相稱呼對方名字的地步。

我該怎麼拒絕他的請求呢?如果説的太直白的話,是不是很容易就把話説死了呢?但是如果不説的話這個問題也着實讓人苦惱,畢竟我不是很想跟他互稱名字著。

"可是阿爾迪王子又説道:"我只是想着今天晚宴上我們已經成為對方的搭檔了,如果還是稱呼王子和公主的話,覺得還是太生疏了,顯得很有距離感,請您放心,我絕對沒有別的意思。"

"嗯...."

他的話聽起來也有那麼幾分道理,他在埃爾米帝國的時候,我將一直是他的搭檔,這種關係也不會隨着稱呼的改變而發生任何變化,也沒什麼問題。

於是我點了點頭。

他看着我,又微笑了起來。"那麼,請叫我艾莎吧,阿爾迪先生。"

聽到我的話,他臉上的笑意更深了。

"好的,艾莎。"

我有些稀裏糊塗的。

"你準備好了嗎?我們出發?"

他笑着伸出手邀請我同行,我自然地握住他的手,一起並肩走向宴會廳。

在走向宴會廳的路上,我在頭腦中把思路又細細地縷了一遍。

今天的宴會絕對不能掉以輕心,雖然只是一個歡迎宴,阿爾迪王子也的確是我的搭檔,但是比起這些,

我還有更重要的事情要做。

但是幸運的是,今天晚上還有許多的朋友來參加這個宴會,這非常令我開心。

如果問我這世界上有什麼令我開心的事情,那當然是和朋友們一起在宴會上玩要了。

一進宴會廳,我就看到了幾張熟悉的臉:羅姿、孔娜、羅蘭,她們都來了。

"大家好久不見!"我朝她們微微一笑,三個人的臉色瞬間變得有些奇怪。

"...嗯?"

好像自從我打扮好之後,所有見到我的人反應都會慢一拍。

我正要説話的時候,羅姿突然驚訝的輕聲喊了一聲:

"艾莎,你今天真的很漂亮。哎呀,快把我眼睛都亮瞎了!"

孔娜和羅蘭也給出了反應:"真的是,我剛剛還以為走進來的是一個精靈呢。"

"過獎了...你們太誇張了..."

我裝作若無其事地謙虛道謝,孔娜卻一直在説:"我們根本沒有過獎。",然後盯着我仔仔細細地看,一副十分亢奮的樣子。

"好啦,別太激動。我來給你們介紹一下俄天公國的王子殿下,阿爾迪·俄天。"

我換了話題,轉身給她們介紹阿爾迪王子。阿爾迪很有禮貌的朝她們點頭致禮。
